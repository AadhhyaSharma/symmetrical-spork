Physics Engine (Python)

A modular 3D physics engine written in Python for experimenting with rigid body dynamics, collision detection, and real-time simulation techniques commonly used in game engines.

Features
3D vector mathematics (Vec3)
Quaternion-based rotations
Rigid body dynamics
Force, torque, and impulse application
Sphere, Box, Capsule, Plane, and Cylinder colliders
Physics material system
Friction and restitution
Sleeping bodies for performance optimization
Raycasting and spatial queries
Fixed timestep simulation
Extensible architecture for future constraints and joints
